# Ignore-Pickup-Items
- The Minecraft mod for **Minecraft 26.1.2**, powered by **NeoForge 26.1.2.68-beta**.
- Requires **Java 25**.
- Ignore items you don't want to pick up — press **(I)** to open the ignore list.

## Build Commands
- `./gradlew --refresh-dependencies`
- `./gradlew clean`
- `./gradlew runClient`
- `./gradlew build`

> The mod version patch number is auto-incremented after each successful build.

## Build Output
After running `./gradlew build`, the mod jar file is generated at:
```
build/libs/MC-Mod-Ignore-Pickup-Items-<version>.jar
```
Copy it to your Minecraft `mods/` folder to install.
