# Auto-Delete-Items
- The Minecraft mod for **Minecraft 26.1.2**, powered by **NeoForge 26.1.2.68-beta**.
- Requires **Java 25**.
- Auto delete selected items from your inventory — press **(O)** to open the config screen and choose items to delete.
- Configure the deletion interval in **Mods → Auto Delete Items → Config**.

## Build Commands
- `./gradlew --refresh-dependencies`
- `./gradlew clean`
- `./gradlew runClient`
- `./gradlew build`

## Build Output
After running `./gradlew build`, the mod jar file is generated at:
```
build/libs/MC-Mod-Auto-Delete-Items-<version>.jar
```
Copy it to your Minecraft `mods/` folder to install.
