# Get EXP From Nature
- The Minecraft mod, NeoForged mc version 26.1.2
- Get 1 exp on every block related to stones break.
- Get 1 exp on every block releted trees break.
- All orb exp works with mending.

- `.\gradlew --refresh-dependencies`
- `.\gradlew clean`
- `.\gradlew runClient`
- `.\gradlew build`
